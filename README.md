# Africa Hardware Plumbing & Electrical - Official Website

A bold, industrial, premium website for Africa Hardware - Alberton's go-to hardware store.

## Features

- **Hero Section** - Bold headline with clear CTAs
- **Why Choose Us** - Feature cards highlighting key benefits
- **Product Categories** - Visual grid of product offerings
- **Google Reviews** - Social proof from real customers
- **Price-Beating Promise** - High-impact guarantee section
- **Store Location** - Embedded Google Maps with store info
- **Sticky Buttons** - WhatsApp and Call buttons for easy contact
- **Mobile-First Design** - Optimized for all devices
- **SEO Optimized** - Local business schema and meta tags

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Inter font (Google Fonts)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
